# Wildcats AI & Cloud Computing Club Website

This repository contains the homepage for the **Wildcats AI & Cloud Computing Club** at Northwest Vista College.

The website serves as the main online presence for the club and provides information about club activities, projects, announcements, and ways for students to get involved.

---

## Project Overview

The Wildcats AI & Cloud Club website is designed to:

- Introduce students to the club and its mission
- Highlight cloud computing and AI projects
- Showcase student work and club activities
- Provide ways for students to join the club
- Share announcements and upcoming events

The site focuses on modern web design with interactive components, animations, and responsive layout for both desktop and mobile users.

---

## Features

- Modern responsive design
- Animated sections using AOS (Animate On Scroll)
- Interactive cards for club activities
- Navigation menu for different site sections
- Call-to-action section for joining the club
- Visitor counter to track site traffic
- Links to future pages including:
  - Guest speakers
  - Game arcade
  - Student animations
  - Featured members

---

## Technologies Used

## Website Sections

The homepage currently includes the following sections:

### Navigation Bar
Provides quick access to different parts of the site.

### Hero Section
Displays the main title and branding for the club.

### About Section
Explains the mission of the Wildcats AI & Cloud Computing Club.

### Experience Our Club
Highlights upcoming pages and interactive experiences including:

- Featured Members
- Guest Speakers
- Game Arcade
- Student Animations

### Join & Feedback
Allows students to:
- Join the club
- Send feedback to the club email

### Footer
Displays page visitor counter and site copyright information.
